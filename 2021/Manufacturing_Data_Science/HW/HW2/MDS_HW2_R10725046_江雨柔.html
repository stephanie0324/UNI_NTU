#### backward elimination
def backward_regression(y,X,initial_list=[],threshold_in=0.01,threshold_out = 0.05,verbose=False):
  included=list(X.columns)
  while True:
      changed=False
      model = sm.OLS(y, sm.add_constant(pd.DataFrame(X[included]))).fit()
      # use all coefs except intercept
      pvalues = model.pvalues.iloc[1:]
      worst_pval = pvalues.max() # null if pvalues is empty
      if worst_pval > threshold_out:
          changed=True
          worst_feature = pvalues.idxmax()
          included.remove(worst_feature)
          if verbose:
              print('Drop  with p-value '.format(worst_feature, worst_pval))
      if not changed:
          break
  return included

backward_regression(y,X)

Q1-D¶

From a methodology aspect, what’s the difference between Ridge regression and Lasso? Why does Lasso support the variables selection rather than ridge? (hint: answer with description or formulation. No computation needed.)

  • 最主要的差異是 ： Lasso 為 Linear Regression 加上 L1，Ridge 為 Linear Regression 加上 L2。 L1 為 絕對值，而L2 則是有平方項的計算。
  • 與Ridge模型最大的差別，就是Lasso會將不具影響力的變數迴歸係數變成0，等於可以自動化的進行變數篩選(Feature selection)。
Q1-e What’s the benefit to use the Elastic Net? (hint: answer with description or formulation. No computation needed.)¶
  • Ridge會有系統的將干擾變數係數逼近0(但不會等於0)，而Lasso則會將較無影響力的變數係數變成0。
  • 而Elastic Net則綜合上述兩者的優勢，同時兼顧有效的正規化過程以及變數挑選功能。

Q1-J¶

What is “adaptive elastic net”? Why we need it? How to build it? Please simply describe or formulate it. (No computation needed.)

ANS:

  • Adaptive elastic net 即多一個變數來調控正規化因子，有點類似 adaptive lasso 加上 elastic net 的形式
  • 使用 adaptive elastic net，是為了減少在高維度產生的偽陽性，同時保持估計的準確性。
  • $\hat{𝛽}(Adapnet) = (1+\frac{\lambda_2}{n})\{argmin ||y-Xβ||^2_2+λ_2||\beta||^2_2+\lambda^*_1\sum_{j=1}^p\hat{w_j}|\beta_j|\}$

Q1-K¶

Which columns are highly-correlated? Show the table of the coefficient estimation by using linear regression. Any multicollinearity problem?
Ans:
以 VIF 方法來找尋具有共線性之變數。首先將每個變數與其他變數算出$R^2$。以導數的方式判斷其大小，VIF越小其共線性越小。以下為其判斷依據。

  • 1 — features are not correlated
  • 1<VIF<5 — features are moderately correlated
  • VIF>5 — features are highly correlated
  • VIF>10 — high correlation between features and is cause for concern

如下所示，透過VIF的計算可看出有出現貢獻性。其中出現貢獻性的變數有：['% Iron Feed', '% Silica Feed', 'Flotation Column 01 Air Flow', 'Flotation Column 03 Air Flow']

from sklearn.linear_model import LinearRegression
def calculate_vif(df, features):    
    vif, tolerance = {}, {}
    no_corr=[]
    mod_corr=[]
    high_corr=[]
    danger_corr=[]
    # all the features that you want to examine
    for feature in features:
        # extract all the other features you will regress against
        X = [f for f in features if f != feature]        
        X, y = df[X], df[feature]
        # extract r-squared from the fit
        r2 = LinearRegression().fit(X, y).score(X, y)                
        
        # calculate tolerance
        tolerance[feature] = 1 - r2
        # calculate VIF
        if(tolerance[feature]!=0):
          vif[feature] = 1/(tolerance[feature])
          if(vif[feature]==1):
            no_corr.append(feature)
          elif(vif[feature]>1 and vif[feature] <5):
            mod_corr.append(feature)
          elif(vif[feature]>5 and vif[feature] <=10):
            high_corr.append(feature)
          elif(vif[feature]>10):
            danger_corr.append(feature)
        
    # return VIF DataFrame
    print('Highly correlated and danger to cause :',danger_corr)

Q1-l Is it possible to predict % Silica in Concentrate without using % Iron Concentrate column (as they are highly correlated)? Why? What’s the potential issue? How to address it?¶

ANS :

  • 可以，使用VIF來計算得出，取任一變數來做計算都不會使其他的變數受到太多的影響。且不被捨棄的變數 VIF也有變小，因此顯示兩者的確有共線性的問題。
  • 可能會遇到知問題：要選擇哪一個捨棄？
  • 解決辦法：
    • 看兩者哪一捨棄使另一方的VIF變得較小來選擇
    • 看哪一項資料取得較容易，就捨棄另一方

def cal_vif_only(df, features):    
    vif, tolerance = {}, {}
    # all the features that you want to examine
    for feature in features:
        # extract all the other features you will regress against
        X = [f for f in features if f != feature]        
        X, y = df[X], df[feature]
        # extract r-squared from the fit
        r2 = LinearRegression().fit(X, y).score(X, y)                
        
        # calculate tolerance
        tolerance[feature] = 1 - r2
        # calculate VIF
        if(tolerance[feature] !=0):
          vif[feature] = 1/(tolerance[feature])
    # return VIF DataFrame
    return pd.DataFrame({'VIF': vif, 'Tolerance': tolerance})
